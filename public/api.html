统一格式：
{
        code: 0成功，非0失败
        msg: 如果code非0，这里是失败理由
        result: {}或者[]，业务字段
}


---------登录接口：user/login     get请求
参数：username, password
返回：realname, company, sid

sid是登录凭证，之后所有接口都传入sid的值


---------注册接口：user/regist   get请求
参数：username, password, realname, company
返回：sid
